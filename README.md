# Azure Security Labs

This repository contains hands-on labs demonstrating various Azure security concepts, architectures, and implementations. Each lab is self-contained with step-by-step instructions, verifications, and architectural insights.

## Labs

1. [Lab 1: Azure RBAC Walkthrough: Users + Groups + Group-Based RBAC (Portal, PowerShell, Bash)](lab-1/README.md)  
   Demonstrates least privilege using group-based access control.

2. [Lab 2: Network Security Groups and Application Security Groups (NSG + ASG)](lab-2/README.md)  
   Enforces least-privilege inbound access using NSGs and ASGs.

3. [Lab 3: Azure Firewall – Controlled Egress Architecture](lab-3/README.md)  
   Centralizes outbound internet traffic inspection with application-level filtering.

4. [Lab 4: Configuring and Securing Azure Container Registry (ACR) and Azure Kubernetes Service (AKS)](lab-4/README.md)  
   Builds and secures container workloads with managed identities and controlled exposure.

5. [Lab 5: Securing Azure Storage with Virtual Network Service Endpoints](lab-5/README.md)  
   Implements network-level protection for Azure Storage using service endpoints.

6. [Lab 6: Azure Monitor Agent (AMA) – Enterprise Monitoring & Security Telemetry Architecture](lab-6/README.md)  
   Deploys centralized telemetry for performance and security monitoring.

7. [Lab 7: Microsoft Defender for Servers Plan 2 – Agentless Enterprise Protection Architecture](lab-7/README.md)  
   Enables agentless protection with FIM and JIT access.

8. [Lab 8: Just-in-Time (JIT) VM Access – Attack Surface Reduction via Dynamic NSG Enforcement](lab-8/README.md)  
   Reduces attack surface with time-bound, IP-scoped access.

9. [Lab 9: Microsoft Sentinel SIEM & SOAR Implementation](lab-9/README.md)  
   Builds an end-to-end SOC pipeline with detection rules and automated response.
